###功能介绍
在线交友婚介系统的web应用  
（1）用户输入帐号,密码登录系统   
（2）注册信息写入到数据库  
（3）浏览首页推荐信息  
（4）搜索好友进行聊天  
（5）添加好友  
（6）个人用户信息管理  
（7）选择群聊空间进行聊天  
（8）聊天内容写入数据库可搜历史记录  
（9）智能聊天对答  
（10）聊天可输入文字和选择表情以及发送图片  
（11）推荐的活动登记和记录  
（12）线上的随机推荐匹配  
（13）用户的管理统计  
（14）用户的签出与注销  
###工具使用
window, node, mongodb
